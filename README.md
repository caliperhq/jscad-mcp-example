# jscad-mcp-example

Demo projects and walkthroughs for [jscad-mcp](https://github.com/caliperhq/jscad-mcp) — an MCP server that gives Claude visual and structural awareness of OpenJSCAD models.

Each demo is designed to showcase a different jscad-mcp feature (standard views, slicing, named parts + highlighting) and to illustrate why the perception loop matters for that class of design problem.
