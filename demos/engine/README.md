# Cutaway 4-stroke engine

Single-cylinder 4-stroke engine, cut away to expose the works. Demonstrates the `slice` MCP tool and multi-file project organization.

See top-level [EXAMPLES.md](../../EXAMPLES.md#engine) for the full walkthrough.

## Files

- `assembly.js` ‚Äî entry point (`main`, `parts`, `getParameterDefinitions`)
- `block.js`, `head.js`, `piston.js`, `conrod.js`, `crankshaft.js`, `valves.js`, `ports.js` ‚Äî per-part modules
- `engine_bundled.jscad` ‚Äî single-file bundle for [openjscad.xyz](https://openjscad.xyz/?uri=https://raw.githubusercontent.com/caliperhq/jscad-mcp-example/main/demos/engine/engine_bundled.jscad). Generated by `../../scripts/bundle-engine.js` ‚Äî do not edit by hand.
